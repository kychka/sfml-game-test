ОПИСАНИЕ ВНУТРЕННЕЙ СТРУКТУРЫ ПРОЕКТА

Дорогой друг! Прежде, чем оставить коммит, убедись, что работаешь с правильной ветвью и внес изменения в правильный каталог. Со списком ветвей и каталогов можно ознакомиться ниже. Будьте внимательны и уважайте других участников!

ВЕТВИ:
master - ветвь релиза. Здесь размещаются только коммиты тимлидера с рабочими версиями проекта.
test - ветвь разработки тренировочного задания.

КАТАЛОГИ:
latest-build - последняя версия сборки всегда тут!
old-builds - старые сборки, не для разработки!
junk - личный мусор участников (используйте для тренировочных коммитов)
contrib - весь код проекта
