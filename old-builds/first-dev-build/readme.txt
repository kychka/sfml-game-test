2-ой путь, для юзеров прочих IDE и ОС. На github.com выложены только исходники, все прочие ресурсы я выкладываю на файлообменник (ссылка для скачивания https://yadi.sk/d/XdupJQEYiFYFV), папку 'resources' необходимо поместить в одну папку с файлом main.cpp, скидываем cpp файлы в проект IDE, компилируем и радуемся =)
3-ий путь, для желающих получить продукт без кода - https://yadi.sk/d/SALyrP3piFYKj, просто скачиваете и запускаете. =)
